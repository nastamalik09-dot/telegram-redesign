# Telegram Home Screen Redesign

Практика Auto Layout, Components и Variants в Figma.

## Задача
Воссоздать домашний экран Telegram с нуля, используя современные возможности Figma.

## Что было сделано
- Создан компонент Chat Row
- Реализованы Variants (Default / Unread)
- Настроен Auto Layout для списка чатов
- Сделаны Header и Bottom Navigation
- Всё собрано в один responsive экран
